### 1.2 常量

这一章的核心是printf输出的格式化字符串。

怎么说呢，格式化字符串的核心其实就是，printf输出的结果一定是一个字符串，然后我们用%c,%d,%f,%ff的规定插入字符串中的格式。

```
printf("character: %c\ninteger: %d\nfloating point: %f\n", '}', 34, 3.14);
```

以上。

|%c|字符串|
|%d|整数|
|%f|浮点数|
|%ff|双精度浮点数|


** Note：如何在C的printf语句中表示"%"? **
```
#include<stdio.h>

int main(){
  printf("百分比符号：%%");
}
```

### 变量声明与变量初始化

和Python不同的是，在C语言中，我们调用一个变量，需要先声明变量类型。

后续如何赋值并没有什么所谓，但是前面一定要声明类型，如：

```
char csb;

int integer;

double floatmax;

float afloat;
```

声明类型后，再赋值的时候，可以不用再复制变量类型。

```
char csb="Chen Sober";

int integer=1;

double floatmax=4.141095;

float afloat=3.0;
```
