## 输入输出 
C++的输出与其他不同。

```
cout << "Hello, World!" << endl;
```

为什么？？
因为cout是C++的输出流，用于输出数据到标准输出设备（通常是屏幕）。

不同在哪里？为什么会有这样的不同？这样为C++带来了什么特性？

带来了C++的流式输出，即可以像cout << "Hello, World!" << endl;这样直接输出字符串，而不用像printf那样用格式化字符串。

纯字符串代码你可能看不出来，但我给你看一个不一样的：

```
int age = 114514
cout <<"Hello, the boy"<<age<<"!"<< endl;
```

这你应该就懂了。

这个<<也许看着很诡异哈，但是这个其实很好记忆的，cout是输出流，把要输出的东西流入输出流，看方向！

而C++的输入流是cin，很自然的，你应该知道方向怎么写了吧？

```
cin >> age;
```

当然，还有另一种方法。

```
#include<simpio.h>
#include<iostream>

using namespace std;

int main(){
  int age;
  age = getInteger("请输入你的年龄：");
  cout << "你好，你今年" << age << "岁了！" << endl;
  return 0;
}
```


simpio.h是一个库，包含了C++的输入输出函数，其主要内容如下表：

|Function name|Description|
|getInteger|获取用户输入的整数|
|getString|获取用户输入的字符串|
|getReal|获取用户输入的浮点数|
|getCharacter|获取用户输入的字符|


另外一提，#include"console.h"帮助我们弹出一个窗口，用于显示输出结果。
