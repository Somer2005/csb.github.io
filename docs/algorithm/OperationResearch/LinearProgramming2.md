# 对偶问题
对偶问题是线性规划中的一个重要概念。每一个线性规划问题都有一个对应的对偶问题。对偶问题的解可以提供原始问题的界限信息，并且在某些情况下，对偶问题的解更容易求解。

## 对偶问题的定义
给定一个标准形式的线性规划问题（称为原始问题）：

* 最大化 \( c^T x \)

* 满足约束条件 \( Ax \leq b \) 和 \( x \geq 0 \)

其对应的对偶问题定义为：

* 最小化 \( b^T y \)

* 满足约束条件 \( A^T y \geq c \) 和 \( y \geq 0 \)

怎么说呢？

首先明确一个口诀吧这是这一章最基础的内容了：

**约束随变量，变量反约束。**

这是什么意思？

对偶问题的约束条件的方向，对应原本该系数的决策变量的正负。

约束问题的变量范围，和这个系数原本对应的右端项的正负，反着来。

## 对偶问题性质

其实是有五个的，但这里只介绍一个：

!!!note "互补松弛性"
    原问题的松弛变量与对偶问题的最优可行解的向量积为零，反之亦然。

影子价格是一个经济学上的概念，这个意思是价值系数（也就是约束条件的右端项）每变化一个，对目标函数会产生多么大的影响。

# 对偶单纯形法
重中之重来了啊。

对偶单纯形法和单纯形法同样重要，对偶单纯形法在后续会被用于求解整数规划的Gomery方法。

对偶单纯形法，其实就是困难一点的单纯形法， 它和单纯形法，主要有两点不同：

* 第一，在我们求解单纯形法的过程中，我们是先用检验数$$C_j-Z_j$$来判别谁入基的，当所有检验数小于等于0的时候，可以判断已达到最优解。
但是，在对偶单纯形法中，我们要先判断谁出基，这个就很好理解嘛，毕竟是对偶单纯形法。

* 第二，对偶单纯形法的右端项不要求为正，且对偶单纯形法的检验数是右端项，当所有右端项大于等于0，此时对偶单纯形法得到最优解，出基的数量为右端项最小的那一个。

* 第三，确定入基变量的方法是计算每一列的检验数比上入基变量行对应的系数值，且只考虑该系数值为负数的值

!!!warning "一定要注意的问题"
    对偶单纯形法是一种单纯形法，本质和大M法，两阶段法没有区别，千万千万千万不要求对偶问题！
