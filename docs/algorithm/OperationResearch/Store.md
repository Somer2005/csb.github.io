# 存储论
抛开那些废话，简单存储论的核心就只有三大补货模型和EOQ。

# 三大补货模型

## t-周期补货模型
t-周期补货模型就是每隔固定时间t进行一次补货，或者说是我们之前学到过的推动型供应链策略饿模型，不管库存水平是多少。这个模型适用于需求比较稳定的商品，比如说日用品。

## $$(S,s)$$补货模型
这个模型的意思是，当库存水平下降到某个下限s时，就会补货到上限S。这个模型适用于需求波动较大的商品，比如说季节性商品或者促销商品，我印象最深的就是盛泽的某个仓库就是这样。

## $$(t，S，s)$$补货模型
这个模型是前两个模型的结合体，就是每隔固定时间t进行一次补货，但是只有当库存水平低于下限s时，才会补货到上限S。这个模型适用于需求既有稳定性又有波动性的商品，比如说电子产品。
